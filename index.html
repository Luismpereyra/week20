Projects

Orphaned Starfish Foundation Re Design

The website was somewhat outdated and lacked a defined brand palette, making it less visually appealing. Our research and user testing revealed that users were frustrated by the limited payment options for donations. Our objective was to design a compelling homepage that evokes emotion, boosts donations, and provides clear transparency on how each contribution supports orphans, while offering multiple payment methods.

Phase 1
Research

Our user research and stakeholder interviews revealed that the top priority for stakeholders was to increase donations, while users expressed a desire for more payment options to facilitate their contributions. To gather these insights, we developed interview plans, conducted interviews with both users and stakeholders, and distributed surveys.

Phase 2
Ideation Phase

We analyzed user insights by graphing data, constructing an empathy map, and building an affinity diagram. We developed user and proto personas, identified key insights, defined main features, and crafted problem statements and value propositions. Subsequently, we designed a user journey map, user flow, and storyboard.
